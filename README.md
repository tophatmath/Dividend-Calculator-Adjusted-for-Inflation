# Dividend-Calculator-Adjusted-for-Inflation
Count the expected nominal of money you will receive in dividends throughout the year with inflation in mind

## Features
- Calculates expected annual dividend income
- Adjusts dividend earnings for inflation
- Supports fractional shares
- Clear, beginner-friendly console interface
- Lightweight and fast

---

## Requirements
- C++ compiler (GCC / MinGW / g++)
- Windows, Linux, or macOS
- Terminal / Command Prompt

---

## How to Compile

Using **g++**:
```bash
g++ DividendInflationCalculator.cpp -o DividendInflationCalculator
